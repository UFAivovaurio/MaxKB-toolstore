# 日期时间戳转换

日期和毫秒级别时间戳的相互转换，如果输入是日期字符串，则转换为毫秒级别时间戳，如果输入是毫秒级别时间戳，则转换为日期。

## 参数说明

### 启动参数    
无

### 输入参数    
| 参数名称 | 参数类型   | 参数说明                                                                                           | 默认值 |
| -------- |--------|------------------------------------------------------------------------------------------------| ------ |
| `input_data` | string | 日期字符串：格式为"YYYY-MM-DD"或"YYYY-MM-DD <br/><br/>HH:MM:SS"（默认视为UTC+8时区），或者毫秒级时间戳，例如"1753795200000"） | |

## 返回说明
- 如果输入是日期字符串，则返回对应的毫秒级时间戳（整数） 
- 如果输入是毫秒级时间戳，则返回对应的UTC+8日期字符串（格式："YYYY-MM-DD HH:MM:SS"）
